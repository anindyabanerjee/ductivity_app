Section 08

    
Build Pipeline

    
From source code to APK on your Android device

  

  

    

      

        💻
        Source Code
        OneDrive directory
      

      

      

        🔃
        Sync to Build Dir
        C:\dev\ductivity_build
      

      

      

        📦
        npm install
        Install dependencies
      

      

      

        📜
        git commit
        EAS needs clean git
      

      

      

        ☁️
        eas build
        EAS Cloud build
      

      

      

        📥
        APK Download
        Install on device
      

    

    

      Why sync to a separate directory? The source lives on OneDrive (C:\Users\anind\OneDrive\Desktop\...) which uses cloud-synced virtual files. EAS Build requires a real git repo with all files physically present. The build directory at C:\dev\ductivity_build is a local-only folder where git and npm can work reliably.
    

    

      .easignore & .gitignore control which files are excluded from the EAS upload and git tracking respectively. node_modules, .expo, and build artifacts are excluded to keep the bundle small and builds fast.
